# Lecture deck → narrated video

Python pipeline for the course assignment: read **`lecture_transcript.txt`** to build **`style.json`**, rasterize **`Lecture_17_AI_screenplays.pdf`**, run Gemini agents for slide descriptions → premise → arc → per-slide narration, synthesize **`audio/slide_NNN.mp3`**, then assemble one **`.mp4`** (basename matches the PDF) with ffmpeg.

## Requirements

- **Python 3.11+**
- **ffmpeg** on your `PATH`
- **Google Gemini API** key

Use the **instructor caption/transcript** required by the course (e.g. the linked *Lecture 11 Section 2* captions) as **`lecture_transcript.txt`**, or split parts as **`lecture_transcript_part*.txt`** in the repo root.

## Environment

Copy **`.env.example`** to **`.env`** in the **repo root** (or create **`.env`** yourself; never commit it):

| Variable | Required | Notes |
|----------|----------|--------|
| **`GEMINI_API_KEY`** | Yes (or `GOOGLE_API_KEY`) | From Google AI Studio / Gemini API |
| **`LECTURE_GEMINI_MODEL`** | No | Default: `gemini-2.5-flash` |
| **`ELEVENLABS_API_KEY`** + **`ELEVENLABS_VOICE_ID`** | No | If set, TTS uses ElevenLabs; else **Edge TTS** (no extra key) |
| **`EDGE_TTS_VOICE`** | No | e.g. `en-US-GuyNeural` |

## Install

```bash
pip install -r requirements.txt
```

**Windows / long paths:** If `pip install` fails under a long OneDrive path, use a venv on a short path (e.g. `C:\ve\hw7-venv`) and run the pipeline with that Python.

## Run

From the **repo root**:

```bash
python run_lecture_pipeline.py --steps all
```

- **`--pdf`** — default: `Lecture_17_AI_screenplays.pdf`
- **`--transcript`** — default: `lecture_transcript.txt` (or split captions as `lecture_transcript_part*.txt`)
- **`--project`** — resume an existing `projects/project_…` folder
- **`--steps`** — comma list: `style`, `raster`, `slides`, `premise`, `arc`, `narration`, `tts`, `video`, or `all`
- **`--force-slides`** — regenerate `slide_description.json`

## Repo layout

| Path | Role |
|------|------|
| `run_lecture_pipeline.py` | Entrypoint |
| `lecture_agents/` | Agents, rasterize, TTS, video helpers |
| `style.json` | Speaking-style profile (regenerated by style step) |
| `lecture_transcript.txt` | Instructor captions for style |
| `projects/` | Timestamped `project_YYYYMMDD_HHMMSS/` outputs |

Do **not** commit generated **`slide_images/`**, **`audio/`**, or **`.mp4`** files (see `.gitignore`).
